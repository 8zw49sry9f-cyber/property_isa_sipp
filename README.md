# Rental Property vs ISA + SIPP Calculator

An interactive investment comparison tool that models keeping a buy-to-let rental property versus selling and investing the proceeds into a GIA, ISA and SIPP.

## Features

- Adjustable assumptions via sliders (property value, growth rates, tax rates, contributions, etc.)
- Configurable time horizon (5–40 years)
- Four chart views: Overview, Property Detail, GIA+ISA+SIPP Detail, Cashflow
- Year-by-year summary table
- Responsive dark-themed UI

## Quick Deploy to StackBlitz

1. Go to [stackblitz.com](https://stackblitz.com)
2. Click **"Start a new project"** → choose **React (Vite)**
3. Replace the default files with the ones in this folder
4. It will auto-install and run — copy the preview URL to share

## Quick Deploy to Vercel

1. Go to [vercel.com](https://vercel.com) and sign up (free) with GitHub
2. Push this folder to a new GitHub repository
3. In Vercel, click **"Add New Project"** → import your repo
4. Leave all settings as default and click **Deploy**
5. You'll get a URL like `your-project.vercel.app` to share

## Run Locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.
